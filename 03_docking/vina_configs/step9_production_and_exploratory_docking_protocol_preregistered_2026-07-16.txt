SPATIAL ORTHOSTERIC FILTER

For each reference ligand and docked pose:
- hydrogen atoms are excluded;
- calculate the unweighted geometric centroid of all heavy atoms;
- calculate the Euclidean distance between the docked-ligand centroid and
  the corresponding crystallographic reference-ligand centroid.

A pose is classified as orthosteric only when:
centroid distance <= 5.0 A.

A pose with centroid distance > 5.0 A is classified as non-specific,
surface-associated, or outside the validated reference region regardless
of its Vinardo score.

The same spatial rule applies to compounds 8, 9, and 11.
No compound-specific exception will be introduced after inspecting results.

POSE COLLECTION

For every ligand-receptor pair:
- perform five independent docking runs;
- retain all 20 modes from every run;
- preserve raw pose files and complete logs;
- calculate score and reference-centroid distance for every mode;
- exclude non-orthosteric poses before consensus analysis.

CONSENSUS CLUSTERING

Orthosteric poses from the five independent seeds will be clustered using:
- heavy atoms only;
- explicit identical atom correspondence within the same ligand;
- direct coordinate RMSD in the fixed receptor coordinate frame;
- no structural superposition before RMSD calculation;
- clustering threshold: RMSD <= 2.0 A.

A consensus cluster must contain poses from at least three of five
independent seeds.

POSE-SELECTION RULE

If at least one consensus orthosteric cluster is present:
1. select the cluster represented in the largest number of independent seeds;
2. if tied, select the cluster with the more favourable median Vinardo score;
3. if still tied, select the cluster with the lower median within-cluster RMSD;
4. select the lowest-scoring pose within the chosen cluster as the
   representative pose.

If no orthosteric cluster is represented in at least three of five seeds:
- no representative pose will be selected;
- the ligand-receptor result will be classified as unstable or
  non-reproducible under the protocol;
- an isolated favourable-scoring pose will not be selected manually.

REPORTING

For each ligand-receptor pair report:
- number of successful runs;
- number of orthosteric poses;
- ranks of orthosteric poses;
- centroid distances from the crystallographic reference ligand;
- consensus-cluster occupancy across seeds;
- representative-pose rank and seed;
- Vinardo score;
- within-cluster RMSD distribution;
- whether the result is reproducible under the consensus criterion.

INTERPRETATION

Vinardo scores are docking scores and must not be described as experimental
binding free energies.

Small score differences must not be used as proof of relative affinity.

Scores must not be compared quantitatively between LasR and PqsR.

LasR docking may be used as validated structural rationalization.

PqsR docking must always be labelled exploratory because confirmatory
redocking validation failed.

PLIP

PLIP analysis will be performed only for the selected representative
orthosteric consensus pose.

No PLIP profile will be generated for:
- non-orthosteric poses;
- isolated poses without consensus support;
- manually selected alternative modes.

MOLECULAR DYNAMICS

Molecular dynamics, RMSD/RMSF trajectories, and MM-GBSA analyses are outside
the scope of the current experimental manuscript and repository.

They are reserved for a separate bioinformatics manuscript and will not be
used to rescue or retrospectively validate the present docking results.
